# 트라이지 라벨

스킬들은 다섯 개의 표준 트라이지 역할을 기준으로 동작합니다. 이 파일은 각 역할을 이 저장소의 이슈 트래커에서 실제로 사용하는 라벨 문자열에 매핑합니다.

| mattpocock/skills 라벨 | 우리 트래커 라벨    | 의미                                       |
| ---------------------- | ------------------- | ------------------------------------------ |
| `needs-triage`         | `needs-triage`      | 메인테이너가 이 이슈를 평가해야 함         |
| `needs-info`           | `needs-info`        | 신고자의 추가 정보 응답 대기               |
| `ready-for-agent`      | `ready-for-agent`   | 완전히 명세화됨, AFK 에이전트 처리 가능    |
| `ready-for-human`      | `ready-for-human`   | 사람의 구현이 필요함                       |
| `wontfix`              | `wontfix`           | 처리하지 않을 항목                         |

스킬이 어떤 역할을 언급할 때 (예: "AFK-ready 트라이지 라벨을 적용") 위 표의 오른쪽 열에 있는 해당 라벨 문자열을 사용하세요.

실제 사용하는 어휘에 맞게 오른쪽 열을 편집하면 됩니다.
